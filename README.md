# Esp01-Pir-Rele

Send udp packet "stato" for active alarm, repeat to disable

Ip: 192.168.4.1
UDP:4210

The relay is connected to the GPIO 0 of the ESP01, while the PIR is connected to the GPIO 2 PIN.
